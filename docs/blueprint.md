# **App Name**: DMC Terminal

## Core Features:

- Dynamic Hero Section: Interactive hero section with a seamless looping WebM video background, a 60% transparent dark overlay, 'typewriter' text loading effect for main titles, a 'green neon glow' effect on the main headline, and dual call-to-action buttons.
- Adaptive Navigation Bar: A fixed-top navigation bar featuring a semi-transparent black 'glassmorphism' style, accented with a 1px neon green separator line at the bottom, and 'terminal code style' logo with anchor links.
- Themed Content Sections: Implement content sections (About, Strategy Advantages, Indicators, Training, Bitunix, Testimonials, FAQ) with alternating dark backgrounds (#050508 and #1A201C) and a downward-scrolling parallax floating wireframe background effect.
- Interactive Trading Concept Display: High-tech data cards with glow-on-hover effects, a simulated K-line chart UI panel with a 'green breathing light' border effect, and a realistic 'terminal UI window' for indicator settings.
- Community Evidence Wall: Display user testimonials and profit screenshots in a Masonry 'waterfall' layout, presenting them as 'Data Packets' with green borders that randomly flash to create a dynamic, data-stream feel.
- AI Trading Explainer Tool: A generative AI tool that can act as a conceptual assistant, taking user queries about trading terms or strategies (like those mentioned on the site) and providing simplified, clear explanations and examples.
- Access Request & Lead Storage: A centralized, 'system access request' terminal dialog for lead generation, featuring blinking input cursors and a prominent 'EXECUTE' button for registration and storing user data (nickname, email, Bitunix UID) in a Firestore database.

## Style Guidelines:

- Primary accent color: Vibrant Neon Green (#0DF258), used for glowing effects, critical text highlights, and interactive elements to signify active states and system messages.
- Background color: Deep charcoal-jade (#111713), forming a rich, dark base for the overall interface, complementing the terminal aesthetic. Section backgrounds will use specific requested dark tones for visual variation.
- Secondary accent color: Luminous Chartreuse (#C8E052), a bright yellowish-green used for subtle highlights, interactive states, and to differentiate key interface elements without overpowering the primary neon green.
- Headline font: 'Space Grotesk' (sans-serif), offering a computerized, techy aesthetic ideal for bold titles and section headings. Body text font: 'Inter' (sans-serif), providing clear readability for longer descriptions and paragraph content with a modern, objective feel. Terminal-style elements and code snippets will use 'Source Code Pro' (monospace) for authenticity.
- Use sleek, minimalist outline icons with a technical or data-driven aesthetic. Incorporate elements like circuit board patterns, abstract data visualizations, or geometric shapes to reinforce the high-tech terminal theme. Icons should have an active 'glowing' state upon interaction.
- A structured, modular layout employing responsive grids (e.g., 3x2 for videos, Masonry for testimonials) and flexible columns. Sections alternate between two deep dark background colors and feature a floating wireframe parallax effect on scroll. Key interactive elements are presented as 'panels' or 'terminal dialogs' for consistency.
- Extensive use of 'glitch', 'typewriter', 'breathing light', and 'scanline' effects. Examples include text loading with a typewriter effect, neon green glowing titles and borders, elements brightening on hover, random flashing of 'data packet' borders, and animated transitions for accordions and input fields (e.g., blinking cursors, expanding text with typewriter effect).
